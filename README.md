# 💼 Job Board — Сайт вакансий на Django

Реалистичный pet-проект сайта вакансий, охватывающий ключевые темы Django: модели, ORM, формы, сигналы, фильтрацию, рассылки и логирование.

---

## 🎯 Цели проекта

- Работодатели публикуют вакансии и получают уведомления об откликах
- Соискатели находят вакансии и откликаются с резюме
- Поиск и фильтрация по городу, зарплате, навыкам и другим параметрам

---

## 🗂 Основные модели

| Модель | Описание |
|---|---|
| `User` | Стандартный Django User, разделён на работодателей и соискателей |
| `Company` | Название, описание, логотип, привязка к работодателю |
| `Vacancy` | Вакансия с зарплатой, городом, навыками и статусом |
| `Skill` | Навык (Python, Django, SQL и т.д.), ManyToMany к Vacancy |
| `Application` | Отклик соискателя: резюме, письмо, статус |
| `Message` | Сообщения между пользователями (опционально) |

---

## 🔄 Ключевые сценарии

1. Работодатель регистрируется → создаёт компанию → публикует вакансию
2. Соискатель находит вакансию → откликается с резюме
3. Работодатель получает уведомление (email + in-app)
4. Соискатель отслеживает статус своего отклика
5. Любой пользователь ищет вакансии по фильтрам

---

## 🛠 Что закреплено в проекте

- **Модели и ORM** — связи, выборки, аннотации
- **Фильтрация и поиск** — по зарплате, городу, навыкам
- **Views (FBV/CBV)** — страницы вакансий, компаний, профилей
- **Формы** — отклик на вакансию, создание вакансии
- **Сигналы** — уведомления при создании откликов
- **Сессии** — последние просмотренные вакансии
- **Logging** — логирование ошибок и действий пользователей
- **Рассылки** — email-уведомления о новых вакансиях и откликах

## ⚙️ Установка и запуск

```bash
git clone <repo-url>
cd job-board

python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```
